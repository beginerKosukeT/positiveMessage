【プログラム概要】
ポジティブなメッセージや元気が出るアドバイスを投稿でき、人工の音声として連続再生すること可能

【使用技術】
バックエンド：Go言語　フロントエンド：JavaScript,HTML, CSS(Bootstrap)　DB：MySQL

【補足】
音声の読み上げ：WebSpeechAPI　デプロイ先：Heroku
